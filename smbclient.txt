smbclient -L //192.168.10.200 -U 'PT\Username' -m SMB3

-L - лист шар для данного айпи (показать доступные шары)
-U - подключиться под пользователем.
-m использовать такой то протокол
-W - для WORKROUP

Две комбинации: -U 'PT\user' либо -U 'user' -W PT


Если DNS резолвят криво, то спасает опция -I
-I айпи адрес
smbclient -L //DC1 -I 192.168.10.200 -U 'PT\user' -m SMB3

Анонимное подключение, вместо опции -U указываем -N (nopass)

Подключение к определенной шаре:
smbclient //192.168.10.200/all_share -U 'Administrator' -W PT
(Указываем именно имя папки к которой хотим подключиться)
Опция -c для автоматизации
